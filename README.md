# Sistema de Generación de Horarios Académicos Inteligente

## 📌 Descripción
Este proyecto consiste en el desarrollo de una aplicación web inteligente que permite generar horarios académicos óptimos considerando múltiples restricciones.

## 🎯 Objetivo
Automatizar la generación de horarios académicos para mejorar la eficiencia y reducir conflictos.

## 🛠 Tecnologías
- React
- Node.js
- MySQL

## 📂 Estructura
- docs: Documentación del proyecto
- frontend: Interfaz de usuario
- backend: Lógica del sistema

- Developers
